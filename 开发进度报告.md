## 已完成功能

### iOS App (阶段一 ~75%)

#### 1. 核心框架
| 模块 | 文件 | 状态 |
|------|------|------|
| App入口 | AppDelegate.swift, SceneDelegate.swift | ✅ |

| 网络层 | WTWAPI.swift | ✅ |
| 存储 | TokenStorage.swift | ✅ |

#### 2. 功能页面

**认证模块**
- LoginViewController - 手机号登录、验证码、微信登录

**新手引导**
- OnboardingViewController - 3页滑动引导

**首页 (Home)**
- HomeViewController - 天气显示、拍照存衣、智能搭配入口

**衣橱 (Wardrobe)**
- WardrobeViewController - 分类筛选、网格展示
- AddClothingViewController - 图片上传、AI抠图、分类选择

**穿搭 (Outfit)**
- OutfitViewController / OutfitResultViewController - 风格切换、穿搭预览、保存/日历
- OutfitCalendarViewController - 月历、穿搭记录

**试穿 (TryOn)**
- TryOnViewController - 3D上身、手势微调（预留接口）

**个人中心 (Profile)**
- ProfileViewController - 菜单列表
- BodyModelViewController - 身型录入（模板/AI/手动）
- MemberCenterViewController - VIP会员、特权、课程
- CoursesViewController - 穿搭课程列表
- SettingsViewController - 设置、退出登录

**AI服务**
- WeatherService - 定位、天气、穿搭推荐
- OutfitGenerationService - 智能搭配生成

## 开发流程

```
代码提交到 GitHub
    ↓
GitHub Actions 自动构建 (macos-14)
    ↓
产出 .ipa 文件
    ↓
下载 IPA → 爱思助手安装到 iPhone
    ↓
测试界面和功能
```
