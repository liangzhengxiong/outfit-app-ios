## 1. MySQL 数据库

### 简介
用于存储用户、衣物、穿搭、日历等核心数据。

### 注册/获取方式

**方式A：本地安装 MySQL**
- 下载地址：https://dev.mysql.com/downloads/mysql/
- 选择 MySQL Community Server
- 安装后创建数据库：`CREATE DATABASE what_to_wear;`

**方式B：使用云数据库**
- 阿里云RDS：https://www.aliyun.com/product/rds/mysql
- 腾讯云DBbase：https://cloud.tencent.com/product/cdb
- 按需选择配置，创建 `what_to_wear` 数据库

### 配置项 (.env)

```env
DATABASE_URL="mysql://用户名:密码@主机地址:3306/what_to_wear"
```

### 初始化命令

```bash
cd services/api
npm run db:generate   # 生成Prisma Client
npm run db:migrate     # 创建数据表
```

## 2. 百度AI 人体分割

### 简介
用于衣物图片AI抠图，移除背景生成透明PNG。

### 注册步骤

**Step 1：注册百度智能云账号**
1. 访问 https://ai.baidu.com/
2. 点击右上角「登录/注册」
3. 可用百度账号直接登录

**Step 2：开通人体分割服务**
1. 登录后进入控制台：https://ai.baidu.com/ai-doc
2. 选择「人体分割」产品
3. 点击「立即使用」
4. 创建应用，填写应用名称（如 `WhatToWear-抠图`）
5. 选择服务类目（图像识别）

**Step 3：获取API Key和Secret Key**
1. 进入「管理应用」页面：https://ai.baidu.com/ai-doc/MEDIAPROCESSING/Xk3bcxb21
2. 找到创建的应用
3. 查看 `API Key` 和 `Secret Key`

### 免费额度
- 每月500次免费调用
- 超出部分按量付费

### 配置项 (.env)

```env
BAIDU_AK_ID=你的API_Key
BAIDU_AK_SECRET=你的Secret_Key
```

### API文档
https://ai.baidu.com/ai-doc/BODY/ek39k5eog

## 3. 微信登录

### 简介
支持用户通过微信账号快捷登录。

### 注册步骤

**Step 1：注册微信开放平台账号**
1. 访问 https://open.weixin.qq.com/
2. 点击「登录」使用微信扫码
3. 完成开发者资质认证

**Step 2：创建移动应用**
1. 进入「管理中心」→「移动应用」
2. 点击「创建移动应用」
3. 填写应用信息：
   - 应用名称：`WhatToWear`
   - 应用图标：上传App图标
   - 应用描述：男生AI智能穿搭APP
4. 平台选择：iOS + Android

**Step 3：获取AppID和AppSecret**
1. 应用创建审核通过后
2. 在应用详情页查看 `AppID` 和 `AppSecret`

### 配置项 (.env)

```env
WECHAT_APP_ID=你的AppID
WECHAT_SECRET=你的AppSecret
```

### iOS配置（Xcode项目）
- 在 Xcode 中添加 URL Scheme：`wxAPPID`（替换为实际AppID）
- 在 Info.plist 添加：
```xml
<key>LSApplicationQueriesSchemes</key>
<array>
    <string>weixin</string>
</array>
```

### Android配置
- 在 `AndroidManifest.xml` 添加：
```xml
<intent-filter>
    <action android:name="android.intent.action.VIEW"/>
    <category android:name="android.intent.category.DEFAULT"/>
    <data android:scheme="wxpAppID"/>
</intent-filter>
```

## 4. 天气API（可选）

### 简介
用于天气穿搭推荐。当前使用mock数据，可选择接入真实天气API提升体验。

### 推荐服务商

**OpenWeatherMap（推荐，免费额度大）**
- 注册：https://openweathermap.org/api
- 免费版：1000次/天
- API文档：https://openweathermap.org/current

**和风天气（国内服务，无需翻墙）**
- 注册：https://id.qweather.com/
- 免费版：1000次/天
- API文档：https://dev.qweather.com/

### 配置项 (.env)

```env
# OpenWeatherMap
WEATHER_API_KEY=你的API_Key

# 或和风天气
HEFENG_KEY=你的API_Key
```

## 环境变量汇总

完成所有配置后，`services/api/.env` 文件应包含：

```env
# 数据库
DATABASE_URL="mysql://用户名:密码@主机:3306/what_to_wear"

# JWT
JWT_SECRET="你的随机字符串（至少32字符）"

# 服务器
PORT=3000

# 百度AI（抠图）
BAIDU_AK_ID=""
BAIDU_AK_SECRET=""

# 微信登录
WECHAT_APP_ID=""
WECHAT_SECRET=""

# 文件存储
UPLOAD_DIR="./uploads"
```
